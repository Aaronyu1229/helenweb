# Helen Web — 台灣品牌出海整合顧問官網

幫台灣中小企業產品打進**北美大型通路（如 Costco、Walmart）**的顧問服務官網。

- 深色高級精品風，11 頁靜態網站（純 HTML + CSS，圖片在 `assets/`）
- 可直接部署到 Cloudflare Pages / Vercel（靜態站，無需建置）

## 頁面

| 檔案 | 內容 |
|---|---|
| `index.html` | 首頁 |
| `about.html` | 關於 |
| `cases.html` | 案例 |
| `services-diagnosis.html` | 服務：產品診斷 |
| `services-expo.html` | 服務：參展 |
| `services-channel.html` | 服務：通路上架 |
| `resources-exhibitions.html` | 資源：展會 |
| `resources-grants.html` | 資源：補助 |
| `resources-blog.html` | 資源：文章 |
| `contact.html` | 聯絡 |
| `boss.html` | 給老闆看 |

## 本機預覽

```bash
python3 -m http.server 8080
# 開 http://127.0.0.1:8080/index.html
```
